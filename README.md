#### 标的数据设计原则

基于滴灌通DRO投资模式核心要点：
1. **DRO (Daily Revenue Obligation)** = 每日收入分成凭证
2. **收入分成模式**：投资方投入资金，从门店日常营业收入中按固定比例分成
3. **三方共管账户**：收入每日归集清分，资金通过三方共管账户管理
4. **行业光谱覆盖**：餐饮、零售、服务、文娱（对应滴灌通27类行业光谱）
5. **投资期限**：通常12-36个月
6. **核心指标**：分成比例、回款周期、回本倍数(MOIC)、IRR

#### 每个标的包含完整字段

- **基础信息**：标的ID、企业名称、统一社会信用代码、联系方式
- **行业定位**：主行业、子行业、地区、城市
- **投资参数**：投资金额、投资期限、分成比例、分成频率
- **财务数据**：日/月/年营收、毛利率、净利率、成本结构
- **回报测算**：年分成收入、预期总回款、回本倍数(MOIC)、IRR
- **保障机制**：履约保证金、最低分成保障、数据透明机制
- **项目材料**：完整的项目文档（品牌介绍、门店信息、经营数据、收入分成机制、风险提示）

## 📝 使用指南

- 中环智能体支持按赛道筛选
- 默认按赛道分组展示，可切换为平铺模式
- 点击智能体卡片编辑配置

### 2. 查看标的项目

- 查看12个不同赛道的标的项目（含3个抖音投流标的）

   - 评估覆盖：合作伙伴推荐、投流效果、品牌评估、风险控制

## 📌 架构说明

### 外环智能体（通用初筛，一票否决）
```
用户 → 负面清单(串行) → 触达审核(串行) → 利益一致性(串行) → 中环
        ↓ 不通过           ↓ 不通过           ↓ 不通过
        终止               终止               终止
```

### 中环智能体（行业深度评估，加权评分）
```
中环入口 → [通用智能体群]          → 综合评分 → 投资建议
        → [赛道专属智能体群]          (加权)
```

### 不同赛道的中环智能体差异

**餐饮赛道**：
- 选址评估（人流量、商圈、竞品）
- 食品安全（资质、供应链、卫生）
- 单店模型（坪效、人效、翻台率）

**零售赛道**：
- 库存管理（周转率、损耗率）
- 供应链效率
- 社区消费分析

**电商赛道**：
- GMV评估（增长、质量）
- 平台关系（多元化、依赖度）
- 主播矩阵（头部依赖风险）

**抖音投流赛道（新增）**：
- 合作伙伴评估（合作时长、累计投流、风控权限）
- 投流效果评估（ROI、GMV、CPM、CTR、完播率）
- 品牌评估（店铺评分、收入占比、增长率、退货率、复购率）
- 风险控制评估（垫资历史、违约记录、押金机制）
