## 项目概述
- **名称**: 滴灌通智能体筛选系统
- **目标**: 构建真正接入大模型的多智能体投资评估系统
- **模式**: 双层漏斗模型（外环一票否决 + 中环加权评分）
- **特色**: 不同赛道拥有专属评估智能体群

## 📌 架构说明

### 外环智能体（通用初筛，一票否决）
```
用户 → 负面清单(串行) → 触达审核(串行) → 利益一致性(串行) → 中环
        ↓ 不通过           ↓ 不通过           ↓ 不通过
        终止               终止               终止
```

### 中环智能体（行业深度评估，加权评分）
```
中环入口 → [通用智能体群]          → 综合评分 → 投资建议
        → [赛道专属智能体群]          (加权)
```

### 不同赛道的中环智能体差异

**餐饮赛道**：
- 选址评估（人流量、商圈、竞品）
- 食品安全（资质、供应链、卫生）
- 单店模型（坪效、人效、翻台率）

**零售赛道**：
- 库存管理（周转率、损耗率）
- 供应链效率
- 社区消费分析

**电商赛道**：
- GMV评估（增长、质量）
- 平台关系（多元化、依赖度）
- 主播矩阵（头部依赖风险）

**抖音投流赛道（新增）**：
- 合作伙伴评估（合作时长、累计投流、风控权限）
- 投流效果评估（ROI、GMV、CPM、CTR、完播率）
- 品牌评估（店铺评分、收入占比、增长率、退货率、复购率）
- 风险控制评估（垫资历史、违约记录、押金机制）
